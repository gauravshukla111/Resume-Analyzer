## 💡 Overview  

Resume Analyzer Pro is a smart web application that analyzes your resume against a job description and gives meaningful insights instead of fake results.

It helps you understand:
- How well your resume matches a job
- Which skills you are missing
- What you should improve

---

## ✨ Features  

- 📄 Upload Resume (PDF)
- 📥 Drag & Drop Support
- 🔍 Resume vs Job Description Comparison
- 🧠 Smart Skill Extraction
- 📊 Real Match Score (No fake 100%)
- ❌ Missing Skills Detection
- 💡 Actionable Suggestions
- 📈 Animated Progress Bar
- 🎨 Premium Dark UI
- 📥 Download Analysis Report (PDF)

---

## ⚙️ Tech Stack  

- HTML5  
- Tailwind CSS  
- JavaScript (Vanilla)  
- PDF.js (Resume Parsing)  
- jsPDF (Report Generation)  

---

## 🧠 How It Works  

1. Upload your resume (PDF)  
2. Paste a real job description  
3. System extracts keywords from both  
4. Matches skills intelligently  
5. Calculates score  
6. Shows missing skills and suggestions

## 🚀 Installation  

Clone the repository:

```bash

```

```bash
open index.html
```

---

## ⚠️ Limitations  

- No backend (frontend only project)  
- No real AI (uses smart logic, not OpenAI API)  
- Accuracy depends on job description quality
